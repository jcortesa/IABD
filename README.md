## 🎯 Descripció del Curs

Aquest repositori conté tots els materials, projectes, exercicis i documentació del **Curs d'Especialització en Intel·ligència Artificial i Dades Massives**, un programa formatiu oficial del sistema educatiu espanyol que forma professionals especialitzats en:

- Desenvolupament de models d'intel·ligència artificial
- Implementació de sistemes d'aprenentatge automàtic
- Programació d'aplicacions d'IA
- Gestió i processament de big data
- Aplicació pràctica de tecnologies de dades massives

### 📚 Informació Oficial

- **Durada**: 720 hores (aproximadament 1 any acadèmic)
- **Modalitat**: Presencial
- **Centre**: IEDIB (Institut d'Estudis de les Illes Balears)
- **Curs Acadèmic**: 2025-2026
- **Informació del Ministeri**: [TodoFP - Especialització en IA i Big Data](https://www.todofp.es/que-estudiar/familias-profesionales/informatica-comunicaciones/ce-inteligencia-artificial-bigdata.html)

## 📖 Mòduls Professionals

### 5071 - Models d'Intel·ligència Artificial
- **Hores**: 165h
- **Descripció**: Fonaments teòrics i pràctics dels models d'IA, algorismes d'aprenentatge, xarxes neuronals, i arquitectures d'intel·ligència artificial.
- **Continguts**:
  - Fonaments de la intel·ligència artificial
  - Algorismes d'aprenentatge supervisat i no supervisat
  - Xarxes neuronals artificials
  - Deep Learning i arquitectures avançades
  - Avaluació i optimització de models

### 5072 - Sistemes d'Aprenentatge Automàtic
- **Hores**: 110h
- **Descripció**: Implementació i desplegament de sistemes de machine learning en entorns reals de producció.
- **Continguts**:
  - Preprocessament i preparació de dades
  - Selecció i enginyeria de característiques
  - Algorismes de classificació i regressió
  - Clustering i reducció de dimensionalitat
  - Validació creuada i mètriques d'avaluació

### 5073 - Programació d'Intel·ligència Artificial
- **Hores**: 180h
- **Descripció**: Desenvolupament d'aplicacions d'IA utilitzant els principals frameworks i llenguatges de programació.
- **Continguts**:
  - Python per a IA i Data Science
  - Frameworks: TensorFlow, PyTorch, Scikit-learn
  - Biblioteques de processament: NumPy, Pandas, Matplotlib
  - APIs i serveis cloud d'IA
  - Desenvolupament d'aplicacions web amb IA

### 5074 - Sistemes de Big Data
- **Hores**: 110h
- **Descripció**: Arquitectures i tecnologies per al processament i emmagatzematge de grans volums de dades.
- **Continguts**:
  - Arquitectures distribuïdes i cloud computing
  - Hadoop ecosystem (HDFS, MapReduce, YARN)
  - Apache Spark per al processament distribuït
  - Bases de dades NoSQL (MongoDB, Cassandra)
  - Streaming de dades amb Apache Kafka

### 5075 - Big Data Aplicat
- **Hores**: 155h
- **Descripció**: Aplicació pràctica de tecnologies de big data en casos d'ús reals i projectes empresarials.
- **Continguts**:
  - Anàlisi de dades massives
  - Visualització de big data
  - ETL i pipelines de dades
  - Data warehousing i data lakes
  - Casos d'ús per sectors (financer, salut, retail, etc.)

## 🚀 Instal·lació i Ús

### Prerequisits
```bash
# Python 3.8 o superior
python --version

# Git
git --version

# Docker (opcional, per a entorns containeritzats)
docker --version
```

### Clonació del Repositori
```bash
git clone https://github.com/jcortesa/IABD.git
cd IABD
```

### Configuració de l'Entorn Virtual
```bash
# Crear entorn virtual
python -m venv iabd_env

# Activar entorn virtual
# Linux/macOS:
source iabd_env/bin/activate
# Windows:
iabd_env\Scripts\activate

# Instal·lar dependències (quan estiguin disponibles)
pip install -r requirements.txt
```

### Estructura de Treball Recomanada
1. Cada mòdul té la seva pròpia carpeta amb materials específics
2. Utilitzar Jupyter Notebooks per a experimentació i prototipat
3. Implementar codi de producció en fitxers `.py` ben estructurats
4. Documentar tots els projectes amb README específics
